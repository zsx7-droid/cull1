## 📸 图片处理要求

### 1. 尺寸优化
使用工具调整到推荐尺寸，避免过大或过小。

**推荐工具**：
- Photoshop
- GIMP (免费)
- Squoosh (在线): https://squoosh.app/

### 2. 格式选择
- **JPG**: 照片类图片（推荐质量80-85%）
- **WebP**: 现代格式，体积更小（强烈推荐）
- **PNG**: 需要透明背景时

### 3. 压缩优化
上传前必须压缩，减小文件体积：

**在线工具**：
- TinyPNG: https://tinypng.com/
- TinyJPG: https://tinyjpg.com/

**目标大小**：
- Banner图: < 500KB
- 商品图: < 300KB
- 文化/活动图: < 300KB

### 4. 命名规范
✅ 正确示例：
```
mountain-song.jpg
colorful-ribbon.jpg
bag-colorful-ribbon.jpg
```

❌ 错误示例：
```
IMG_20240115.jpg
新建图片.jpg
DSC0001.JPG
```

1. 准备好所有图片文件

- ✅ Banner轮播正常显示

1. 检查文件名是否完全匹配（区分大小写）
2. 检查文件是否在正确的目录

## 🔧 技术说明

### 引用方式
```vue
<!-- 绝对路径引用（推荐） -->
<img src="/images/culture/mountain-song.jpg" alt="畲族山歌" />

<!-- 动态绑定 -->
<img :src="item.coverImage" :alt="item.title" />
```

### 路径规则
- `/images/` 对应 `public/images/` 目录
- 不需要写 `public` 前缀
- 必须以 `/` 开头
