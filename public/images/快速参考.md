## ⚠️ 注意事项

1. **文件名必须完全匹配**（包括大小写）
2. **图片格式支持**: JPG, PNG, WebP, GIF
3. **建议压缩图片**以提升加载速度
4. **刷新浏览器**查看最新效果

## 🔍 验证步骤

1. 将图片放入对应目录
2. 刷新浏览器 http://localhost:3000
3. 按 F12 打开开发者工具
4. 查看 Network 标签，确认图片加载成功（状态码200）
5. 检查页面是否正常显示所有图片

## 📞 常见问题

**Q: 图片显示不出来怎么办？**
A: 
1. 检查文件路径是否正确
2. 检查文件名是否完全匹配
3. 查看浏览器Console是否有404错误
4. 确认图片文件格式正确

**Q: 如何批量重命名图片？**
A: 
- Windows: 使用 PowerToys PowerRename
- Mac: 使用 Finder 批量重命名
- 在线工具: https://bulkrenameutility.co.uk/

**Q: 图片太大影响加载速度？**
A: 
- 使用 TinyPNG 压缩: https://tinypng.com/
- 转换为 WebP 格式
- 调整到合适的尺寸
